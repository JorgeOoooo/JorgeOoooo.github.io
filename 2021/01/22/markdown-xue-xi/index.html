Markdown 学习

Markdown
发布日期:   2021-01-22

基于Markdown编辑器Typora。

多级标题

输入：#*n+空格

输入快捷键：Ctrl+1~6

清除快捷键：Ctrl+0

注意：

  1. 编号后，一定要有空格！
  2. #个数代表不同级别的标题
  3. 1 <= n <= 6

有序列表

样例：
  1. 有序列表
    1. 有序列表
    2. 有序列表
  2. 有序列表
  3. 有序列表
输入：编号+.”+空格

输入快捷键：Ctrl+Shift+[

降级快捷键：Tab

升级快捷键：Shift+Tab

注意：

  1. 编号后，一定要有空格！
  2. 行末单击回车，序号会自动添加。
  3. 添加/删除某一行内容，序号会自动调整。
  4. 行末向下键/双击回车，序号不会添加。

无序列表

样例：
  • 无序列表
    • 无序列表
    • 无序列表
  • 无序列表
  • 无序列表
输入：*或-+空格

输入快捷键：Ctrl+Shift+]

降级快捷键：Tab

升级快捷键：Shift+Tab

注意：

  1. 编号后，一定要有空格！
  2. 行末单击回车，无序列表会自动添加。
  3. 行末向下键/双击回车，序号不会添加。
  4. 有序列表和无序列表可以交错使用。

任务列表

样例：
  • 任务一
  • 任务二
  • 任务三
输入：*或-+空格+[+空格+]+空格

注意：

  1. 符号后一定要加空格！
  2. 行末单击回车，列表会自动添加。
  3. 行末向下键/双击回车，序号不会添加。
  4. 无快捷键。

表格

样例：
表头 表头 表头 表头
信息 信息 信息 信息
信息 信息 信息 信息
输入：|+表头+|+表头+|+回车

输入快捷键：Ctrl+T

添加行：Ctrl+回车

注意：

  1. 表头输入完成，回车生成表格。
  2. 行内容中仅单击回车，不会添加行。
  3. 对齐方式、其他操作可通过Typora中的按钮添加。

代码块

样例：
public class helloWorld{
    public static void main(String[] args){
        System.out.pringln("Hello World");
    }
}
输入：```+语言类型+回车

输入快捷键：Ctrl+Shift+K

区块

样例：

一

二

三

输入：>+空格

注意：

  1. 区块可嵌套。
  2. 列表可嵌套。

链接

样例：

这是一个链接。

输入：[+内容+]+(+链接+)

输入快捷键：Ctrl+K

图片

样例：

输入：![+内容+]+(+链接+) 或 <img src="" width="50%">

输入快捷键：Ctrl+Shift+I

行内代码

样例：

java是一门面向对象编程语言。

输入：`+内容+`

输入快捷键：Ctrl+Shift+`

其他

目前支持的 HTML 元素有：<kbd> <b> <i> <em> <sup> <sub> <br>等。
